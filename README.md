# Instagram Card Public Assets

이 저장소는 private 자동화 저장소 `aiaxdev/instagram-card`의 GitHub Actions가 생성한 인스타그램 카드 이미지를 공개 URL로 제공하기 위한 public repository입니다.

## 현재 구조

```text
cards/
  01.jpg
  02.jpg
  03.jpg
  04.jpg
  05.jpg
  06.jpg
  07.jpg
  08.jpg
  09.jpg
  10.jpg
README.md
```

## 사용 URL

Instagram Content Publishing API에서 접근하는 공개 이미지 URL 형식입니다.

```text
https://raw.githubusercontent.com/aiaxdev/instagram-card-pub/main/cards/01.jpg
...
https://raw.githubusercontent.com/aiaxdev/instagram-card-pub/main/cards/10.jpg
```

이 저장소는 수동 편집하지 않고 GitHub Actions 결과물로 갱신하는 것을 기준으로 합니다.
